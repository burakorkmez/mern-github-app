### Setup .env file

```js
PORT=5000
MONGO_URI=
GITHUB_API_KEY=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
CLIENT_BASE_URL=
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
